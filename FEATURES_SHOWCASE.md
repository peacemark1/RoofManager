## Animation Principles

### Entrance Animations (Page Load)
- Staggered start times (0.2s - 0.8s)
- Fade in with Y translation
- Duration: 0.6 - 0.8 seconds
- Easing: ease-out

### Scroll Animations
- Triggered by `whileInView`
- Fade in when scrolled into viewport
- Duration: 0.6 seconds
- Once: true (only animate once)

### Interactive Animations
- Hover: scale 1.02 - 1.1, duration 0.3s
- Tap: scale 0.95, duration instant
- Focus: border color change, glow effect

### Continuous Animations
- Background orbs: infinite scale/position loops
- Duration: 8 - 20 seconds
- Easing: ease-in-out
